# SAMSTAGRAM
Samstagram is a console-based social networking application developed in C that simulates the core functionalities of a social media platform using efficient data structures and algorithms.

It is a group project which is developed in C language using data structures as a console based project that stimuluates basic functionalities of a social media platform like instagram 
User can add friends , view friends , get friend suggestions , see mutuals and some other basic functions like viewing the user with highest functions 
The project is developed using the data structues like Trees , Tries and also graphs are implemented in the project. 
The project also suppourts basic authentication for the user
